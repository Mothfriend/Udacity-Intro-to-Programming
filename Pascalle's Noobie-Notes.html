Intro to the World Wide Web

The Basics of the Web

The main type of document on the web is HTML. Different types of files also exist on the web as plain text, HTML, images, videos, and audio.

The main components of the Web are:

  • a browser: application that interprets files exchanged between servers and devices
  • internet: network of computers all linked together so they can exchange files
  • HTTP: Hypertext Transfer protocol is a mechanism used for communitcation between browsers and servers
  • servers: computers that host files to be accessed by a browser

Basics of HTML

HyperText Markup Language is a document written in language recognized by browsers. HTML elements consist of opening and closing tags that wrap the content of a webpage within the document. These tags instruct a web browser to display content a certain way on a web page. Some elements can have attributes assigned to them to manipulate how the content behaves. Example: "href" is an attribute that defines a URL to a link.

In these notes, elements are used to create bold headers and subheaders, define paragraphs, and source image content.

Container elements create a box around the content. Usually block elements are coded at the beginning of a new line in the text editor. Inline, however, are container elements written within the a line of code to isolate parts of content within the same line in an editor.

Span and Div are two examples of inline and block elements. Span, being inline, will control how isolated text within a line displays or behaves. Div, blocks out content for the purpose of applying layout styles to a section that's been blocked out.
